Problem Statement

PRINT MASTER 

 

Generally, a client visits a print house with the specifications for a print job and expects a quote or an estimate for the proposed print job. The print house, based on these specifications generate a quote for the client which can be printed out in hardcopy and or is sent to the client via mail for approval, a digital copy of this quote should be saved for future referencing. On obtaining said approval, a work order is generated to be sent to the print floor. This work order should also be printed out in hardcopy format, a copy of this work order should be saved for later reference.

 

Data concerning available machinery on the print / work floor should be documented and also saved on the system since various parameters from these print machines are required to aid in generating quotes.

Various materials are needed for day to day printing jobs in a print house like papers, inks etc and should be documented and updated in an inventory.

 

Focus will be placed on the following set of requirements while designing the system. 

  1. A user in this context could either be an individual or a company and should be able to either create an account and modify an account on the system
  2. Any user on the system should be able to create quotes, modify quotes, print quotes, and send quotes (via email).
  3. A user should be able to search for quotes by client name, quote number or by client contact number.
  4. Each quote will have a uniquely identifying number and other details like; customer name, job description, date of creation, finished and press formats, inks for both inserts and covers etc.
  5. A user on the system should be able to create work orders, modify the order, delete the order as well as print (hardcopy format) work orders as well as file requests and or reports on work orders from the print floor(i.e. when the order makes it to the print floor).
  6. A user should be able to search for work orders by order id, client name or the associated quote.
  7. A user should be able to view the full details of any quote or work order.
