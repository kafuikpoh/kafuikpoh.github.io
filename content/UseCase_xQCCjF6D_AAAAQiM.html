Scenario
Steps
1. User indicates desire to create a user account and enters basic information.
    1.1. SYSTEM creates a new user.
    1.2. SYSTEM prompts for user addresses.
2. User enters one or more addresses.
    2.1. SYSTEM creates addresses.
    2.2. SYSTEM prompts for company details.
3. User enters company information.
    3.1. SYSTEM creates company profile.
    3.2. prompts for Credit/debit card or MoMo.
4. User enters credit/debit card or MoMo information.
    4.1. SYSTEM creates account.
    4.2. SYSTEM verifies authorization for credit/debit card.
    4.3. SYSTEM associates company, address, and account.
    4.4. SYSTEM returns valid customer account details.

Extension
Steps
1.a. Basic user data are incomplete
    1. SYSTEM highlight fields and prompt: "This data is required. Cannot be empty".
4.a. Credit/debit or MoMo information isn't valid.
    1. SYSTEM highlight field and prompt: "Credit/debit card number isn't valid".
    2. SYSTEM highlight field and prompt: "MoMo number isn't valid".
